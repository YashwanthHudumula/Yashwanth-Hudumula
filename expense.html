Expense Management System

A user-friendly web application for tracking and managing expenses

Project Overview

This project was developed as part of my internship at Infosys. It provides an efficient system for tracking, managing, and analyzing expenses to ensure transparency and automation in financial management.

Key Features

  • Secure user registration and role-based dashboard access
  • Interface for adding, updating, and managing expenses
  • AI-powered categorization of expenses
  • Real-time notifications and secure backend management

Technologies Used

  • Frontend: React
  • Backend: Spring Boot
  • Database: MySQL
  • Cloud Hosting: Microsoft Azure
